# React

## 环境

```bash

#全局安装脚手架工具
npm install -g create-react-app
create-react-app 项目名称

#可以使用npx直接创建react项目
npx create-react-app 项目名称 

#创建react+typescript项目
npx create-react-app 项目名称 --template typescript

cd 项目名称
#安装项目依赖
npm install
#运行项目
npm start

```
