# TypeScript

## TypeScript环境

```bash

#全局安装TypeScript
npm install -g -typescript

#查看TypeScript版本
tsc -V

#运行ts文件
tsc xx.ts #xx.ts是要运行的ts文件

#运行编译号的js文件
node xx.js #xx.js是上一步操作后产生的.js文件

#安装轻量级运行环境lite-server
npm install --save-dev lite-serve

#编译环境的ES版本更换
tsc -t esx xx.ts #esx是ES版本

```
